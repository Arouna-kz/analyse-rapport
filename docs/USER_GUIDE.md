## Introduction

Bienvenue sur la Plateforme d'Analyse IA ! Cette application vous permet d'analyser vos rapports avec l'intelligence artificielle, de générer des prédictions et d'obtenir des insights automatiques.

### Fonctionnalités principales

- **Analyse automatique** : Extrayez les KPIs et insights de vos documents
- **Génération IA** : Créez des rapports à partir de modèles ou données existantes
- **Génération depuis un modèle** : Reproduisez la structure d'un rapport existant avec de nouvelles données
- **Chat IA** : Posez des questions sur vos rapports avec support de fichiers et images joints
- **Arena Multi-Modèles** : Obtenez des réponses consensus de plusieurs IA simultanément
- **Prédictions** : Générez des scénarios optimistes, réalistes et pessimistes
- **Alertes** : Recevez des notifications sur les anomalies détectées
- **Export** : Téléchargez vos analyses en Excel ou PDF
- **Interactivité** : Affinez les résultats avec vos retours personnalisés
- **Historique des versions** : Consultez les différentes versions de vos analyses

## Premiers pas

### Création de compte

1. Accédez à la page d'accueil
2. Cliquez sur "Commencer gratuitement"
3. Remplissez le formulaire d'inscription avec votre email et mot de passe
4. Vérifiez votre boîte mail et confirmez votre compte

### Connexion

1. Cliquez sur "Se connecter"
2. Entrez vos identifiants
3. Vous serez redirigé vers le tableau de bord

### Réinitialisation du mot de passe

1. Sur la page de connexion, cliquez sur "Mot de passe oublié ?"
2. Entrez votre adresse email
3. Consultez votre boîte mail pour le lien de réinitialisation
4. Définissez un nouveau mot de passe

### Changement de mot de passe

1. Depuis le tableau de bord, cliquez sur l'icône de profil
2. Sélectionnez "Changer le mot de passe"
3. Entrez votre **mot de passe actuel** pour vérification
4. Saisissez et confirmez votre nouveau mot de passe
5. Cliquez sur "Changer le mot de passe"

## Tableau de bord

Le tableau de bord est votre centre de contrôle. Vous y trouverez :

### Vue d'ensemble

- **Nombre total de rapports** : Compteur de tous vos documents
- **Analyses complétées** : Nombre d'analyses terminées
- **Alertes actives** : Notifications nécessitant votre attention

### Actions rapides

- **Importer des rapports** : Uploader un nouveau document
- **Générer avec IA** : Créer un rapport futur basé sur vos données
- **Générer depuis modèle** : Créer un rapport en s'inspirant d'un modèle existant
- **Générer prédiction** : Créer des scénarios prédictifs

- **Voir les alertes** : Consulter les notifications
- **Configuration Arena** : Configurer les modèles IA multi-consensus

### Liste des rapports

Vous pouvez voir tous vos rapports organisés par onglets :
- **Passés** : Rapports historiques
- **Actuels** : Rapports en cours
- **Futurs** : Rapports générés par l'IA

Chaque rapport affiche :
- Titre du rapport
- Type (passé, présent, futur)
- Statut (en attente, en cours, terminé, erreur)
- Date de création
- Actions disponibles (télécharger, supprimer, ré-analyser)

## Upload de rapports

### Formats supportés

- **PDF** : Documents au format Portable Document Format
- **DOCX** : Documents Microsoft Word
- **TXT** : Fichiers texte simple
- **XLSX** : Fichiers Excel (format moderne)
- **XLS** : Fichiers Excel (format classique)

### Comment uploader

1. Cliquez sur "Importer des rapports" dans le tableau de bord
2. Cliquez sur la zone d'upload ou glissez-déposez votre fichier
3. Donnez un titre à votre rapport
4. Sélectionnez le type de rapport :
   - **Passé** : Données historiques
   - **Présent** : Données actuelles
   - **Futur** : Projections (généré par l'IA)
5. Cliquez sur "Uploader et analyser"

### Prévisualisation du contenu extrait

Avant de lancer l'analyse, vous pouvez prévisualiser le contenu extrait de votre document pour vérifier la qualité de l'extraction.

### Suivi du traitement

Une barre de progression vous indique l'état de l'analyse :
- **En attente** : Le fichier est en file d'attente
- **En cours** : L'IA analyse votre document
- **Terminé** : L'analyse est disponible
- **Erreur** : Un problème est survenu

## Analyse des rapports

### Consulter une analyse

1. Cliquez sur un rapport dans la liste
2. Vous verrez :
   - **Résumé** : Vue d'ensemble générée par l'IA
   - **Points clés** : Éléments importants identifiés
   - **KPIs** : Indicateurs de performance extraits avec graphiques
   - **Insights** : Recommandations et observations
   - **Graphiques** : Visualisations des données (barres, cercles, lignes)
   - **Score Arena** : Score de consensus multi-modèles (si activé)

### Améliorer l'analyse (Interactif)

Si l'analyse ne correspond pas à vos attentes :
1. Scrollez jusqu'à la section "Améliorer l'analyse"
2. Décrivez ce que vous souhaitez voir (ex: "Plus de détails sur les KPIs financiers")
3. Vous pouvez joindre des fichiers ou images supplémentaires via les icônes 📎 et 🖼️
4. Cliquez sur "Relancer l'analyse"
5. L'IA générera une nouvelle analyse tenant compte de vos retours

### Historique des versions

Chaque amélioration d'analyse est sauvegardée dans l'historique :
- Consultez les versions précédentes
- Comparez les résultats entre versions
- Restaurez une version antérieure si nécessaire

### Graphiques interactifs

Les graphiques peuvent être :
- **Agrandis** : Cliquez sur l'icône d'expansion pour un affichage plein écran
- **Téléchargés** : Exportez en image
- **Partagés** : Générez un lien de partage

### Export de l'analyse

- **Excel** : Téléchargez les données brutes structurées
- **PDF** : Exportez un rapport formaté avec graphiques

## Génération depuis un modèle

Cette fonctionnalité permet de créer un nouveau rapport en reproduisant la structure et le style d'un rapport existant.

### Accéder à la génération depuis modèle

1. Cliquez sur "Générer depuis modèle" dans le tableau de bord
2. Ou accédez à `/generate-template` via le menu

### Étapes de génération

#### Étape 1 : Importer les données

1. Uploadez votre fichier de données (PDF, DOCX, TXT, XLSX, XLS)
2. Ce fichier contient les informations brutes à analyser
3. Taille maximale : 20 Mo

#### Étape 2 : Choisir la source du modèle

Vous avez deux options :

**Option A - Fichier modèle :**
- Uploadez un document qui servira de modèle
- L'IA reproduira la structure et le style de ce document

**Option B - Rapports existants :**
- Sélectionnez un ou plusieurs rapports de votre base
- L'IA s'inspirera de leur format et structure

#### Étape 3 : Personnaliser

1. Donnez un titre à votre nouveau rapport
2. Ajoutez des instructions supplémentaires (optionnel) :
   - "Focus sur les indicateurs financiers"
   - "Inclure une comparaison avec le trimestre précédent"
   - "Mettre en avant les points d'amélioration"

#### Étape 4 : Générer

1. Cliquez sur "Générer le rapport"
2. L'IA analyse vos données et crée le rapport
3. Le traitement peut prendre quelques minutes

### Amélioration interactive

Après génération, si le résultat ne vous convient pas :
1. Accédez au détail du rapport généré
2. Utilisez la section "Améliorer l'analyse"
3. Précisez ce que vous souhaitez modifier
4. Joignez des fichiers ou images si nécessaire
5. L'IA régénérera le contenu selon vos indications

## Chat IA

### Démarrer une conversation

1. Cliquez sur "Chat IA" dans le menu
2. Posez votre question dans le champ de texte
3. L'assistant répond en utilisant vos rapports comme contexte (recherche sémantique RAG)

### Gestion des conversations

- **Nouvelle conversation** : Cliquez sur "Nouvelle conversation" dans le panneau latéral
- **Historique** : Retrouvez toutes vos conversations précédentes dans le panneau latéral
- **Renommer** : Modifiez le titre d'une conversation
- **Supprimer** : Supprimez une conversation et ses messages

### Joindre des fichiers

Vous pouvez enrichir vos questions avec des fichiers :
1. Cliquez sur l'icône 📎 pour joindre un fichier (PDF, DOCX, Excel, TXT)
2. Cliquez sur l'icône 🖼️ pour joindre une image
3. Maximum 5 fichiers par message
4. Les fichiers sont analysés par l'IA pour enrichir le contexte

### Exemples de questions

- "Quels sont les principaux KPIs de mon dernier rapport ?"
- "Compare les performances entre janvier et février"
- "Quelles tendances observes-tu dans mes données ?"
- "Recommande des actions basées sur mes analyses"

### Bonnes pratiques

- Soyez précis dans vos questions
- Référencez les périodes ou rapports spécifiques
- Joignez des fichiers pour plus de contexte
- Demandez des clarifications si nécessaire

## Prédictions multi-scénarios

### Générer des prédictions

1. Accédez à "Prédictions" dans le menu
2. Sélectionnez les rapports de base (minimum 1)
3. Cliquez sur "Générer les scénarios"

### Types de scénarios

- **Optimiste** : Projections favorables
- **Réaliste** : Projections équilibrées
- **Pessimiste** : Projections prudentes

### Améliorer les prédictions (Interactif)

Si les prédictions ne vous satisfont pas :
1. Scrollez jusqu'à "Améliorer cette prédiction"
2. Décrivez vos attentes (ex: "Projections plus détaillées pour le CA")
3. Joignez des fichiers ou images si nécessaire
4. Cliquez sur "Affiner les prédictions"

### Sauvegarder et partager

1. Donnez un nom à votre scénario
2. Cliquez sur "Sauvegarder"
3. Utilisez le bouton "Partager" pour générer un lien

### Comparer les scénarios

- Ouvrez l'historique des prédictions
- Sélectionnez plusieurs scénarios (2-3)
- Utilisez la vue de comparaison côte à côte

## Alertes

### Types d'alertes

- **Seuil dépassé** : Un KPI dépasse une limite définie
- **Anomalie détectée** : Valeur inhabituelle identifiée
- **Inversion de tendance** : Changement de direction significatif
- **Données manquantes** : Informations absentes
- **Problème de qualité** : Incohérences détectées

### Niveaux de sévérité

- 🟢 **Faible** : Information à noter
- 🟡 **Moyenne** : Attention requise
- 🟠 **Élevée** : Action recommandée
- 🔴 **Critique** : Action urgente requise

### Gérer les alertes

1. Accédez à la page "Alertes"
2. Consultez les détails de chaque alerte
3. Marquez comme "Reconnu" une fois traité
4. Les alertes sont présentées dans une liste scrollable

## Système Arena Multi-Modèles

### Qu'est-ce que l'Arena ?

L'Arena est un système avancé qui interroge **plusieurs modèles d'IA en parallèle** pour produire une réponse consensus optimale appelée "Réponse Gold". Ce mécanisme réduit les erreurs et hallucinations en croisant les perspectives de différents modèles.

### Comment ça fonctionne

1. Votre requête est envoyée simultanément à plusieurs modèles IA
2. Chaque modèle génère sa propre réponse indépendamment
3. Un modèle "Juge" analyse toutes les réponses
4. Le Juge produit une synthèse optimale (Réponse Gold) et détecte les hallucinations
5. Un score de consensus indique le niveau d'accord entre les modèles

### Modèles disponibles (Lovable AI)

- **Gemini 2.5 Pro** : Modèle haut de gamme pour le raisonnement complexe
- **Gemini 2.5 Flash** : Modèle rapide et équilibré
- **GPT-5** : Modèle puissant polyvalent
- **GPT-5 Mini** : Modèle économique avec bonnes performances

### Configuration de l'Arena

1. Accédez à "Configuration Arena" depuis le tableau de bord
2. Activez/désactivez les modèles souhaités
3. Choisissez le modèle Juge
4. Vous pouvez aussi ajouter des modèles externes compatibles OpenAI
5. Sauvegardez votre configuration

### Indicateurs Arena

- **Score de consensus** : Pourcentage d'accord entre les modèles
- **Hallucinations détectées** : Incohérences identifiées et filtrées
- **Temps de traitement** : Durée de l'analyse multi-modèles
- **Réponses individuelles** : Accès aux réponses de chaque modèle (Mode Expert)

## Partage et export

### Partager une analyse

1. Ouvrez le rapport ou la prédiction
2. Cliquez sur l'icône de partage
3. Copiez le lien généré
4. Définissez une date d'expiration (optionnel)

### Formats d'export

- **Excel (.xlsx)** : Données structurées dans des feuilles
- **PDF** : Document formaté avec graphiques

### Télécharger un rapport

1. Dans le tableau de bord, cliquez sur l'icône de téléchargement
2. Le fichier original sera téléchargé

### Supprimer un rapport

1. Dans le tableau de bord, cliquez sur l'icône de suppression
2. Confirmez la suppression

### Suivi des partages

Consultez les statistiques de vos liens partagés :
- Nombre de vues
- Localisation des visiteurs
- Date de dernière consultation
